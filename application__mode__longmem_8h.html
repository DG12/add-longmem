Detailed Description

override default configuration from app_config for longmem variant of firmware. Increasing the interval between storing sensor readings(history) to save as many as possible. Don't compile sensor modules for sensors that are not installed. ToDo change loader directives to lower the flash boundry and increase pages for history.
